# ストレスチェック質問テンプレート

厚生労働省「職業性ストレス簡易調査票（57項目）」に準拠

## 形式

CSVアップロード時の列形式:
```
order,section,text,reverse
```

- `order`: 質問番号（1-57）
- `section`: セクション（A/B/C/D）
- `text`: 質問文
- `reverse`: 逆転項目（true/false）。trueの場合 score = 5 - 回答値

## セクション定義

| セクション | 内容 | 問数 |
|---|---|---|
| A | 仕事のストレス要因 | 17問（1-17） |
| B | ストレス反応 | 29問（18-46） |
| C | 周囲のサポート | 9問（47-55） |
| D | 満足度 | 2問（56-57） |

## 回答形式

4択（1〜4）:
1. そうだ
2. まあそうだ
3. ややちがう
4. ちがう

## 高ストレス判定

合計スコア **77点以上** を高ストレスと判定
（逆転項目: score = 5 - 回答値 として加算）

## CSVアップロード例

```csv
order,section,text,reverse
1,A,非常にたくさんの仕事をしなければならない,false
2,A,時間内に仕事が処理しきれない,false
8,A,自分のペースで仕事ができる,true
18,B,活気がわいてくる,true
```
